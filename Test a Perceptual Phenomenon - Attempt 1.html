Analyzing the Stroop Effect¶

Perform the analysis in the space below. Remember to follow the instructions and review the project rubric before submitting. Once you've completed the analysis and write up, download this file as a PDF or HTML file and submit in the next section.

(1) What is the independent variable? What is the dependent variable?

Independent variable 1 - the color of the ink matches the printed words (Congruent Word List).

Independent variable 2 - the color of the ink is different than the printed words (Incongruent Word List).

Dependent variable - time to name the color of the ink in which the words are printed.

(2) What is an appropriate set of hypotheses for this task? What kind of statistical test do you expect to perform? Justify your choices.

$H_{0}$: $\mu_{c} \geq \mu_{i}$

Null Hypothesis: the average time for the congruent word list is greater than or equal to the average time for the incongruent word list.

$H_{A}$: $\mu_{c} < \mu_{i}$

Alternative Hypothesis: the average time for the congruent word list is less than the average time for the incongruent word list.

The type of test to use is a one-tailed t-test [1]. This type of test is the best fit since we are looking to find out if times are greater with the incongruent words than with the congruent words.

import scipy.stats as stats
